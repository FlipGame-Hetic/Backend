## Skills Bonus (Defensive/Utility Skills)

| Name                          | Description                                            | Duration/Effect                 | Notes                                                                                            |
| --------------------------- - | ------------------------------------------------------ | -------------------------- - | ------------------------------------------------------------------------------------------------ |
| **Shield**                 | Temporary protection that prevents the loss of a ball | 10–30 seconds              | Disappears as soon as a ball is lost; blocks ultimate energy build-up whilst active |
| **Time Slowdown**  | Slows the ball’s speed                        | ~ 5–10 seconds             | Linked to the character with an invisible penalty to balance the game                                            |
| **Combo Multiplier**  | Multiplies points scored (x2 to x4)                 | Variable duration              | ‘Hacker’ mode; rewards skilful play (for skilled players like ‘Jett’)                        |
| **Damage/Points Boost**   | Basic increase in damage dealt               | Standard                    | Simple but effective 
| **Extra flippers** | Adds 2 flippers (increases from 4 to 6)                     | Temporary                  | Works in parallel with cybernetic bioaugmentation                                                      |
| **Portals/Teleportation**   | Places a checkpoint and allows you to go back         | 2 charges: entry + exit | Retains points earned; inspired by Portal/Apex Legends                                     |
| **Freeze/Ice Effect**       | Freezes the opponent momentarily                       | Short duration                | Sliding effect afterwards (increases speed); the ball stops then starts moving again                        |
| **Extra Ball**     | Adds a temporary ball                            | Single use                | Bonus from the pinball machine itself (not a character ability); must be limited to prevent abuse          |

## Skills (Offensive Abilities)

| Name                      | Description                                      | Impact   | Notes                                                                   |
| ------------------------ | --------------------------------------- --------- | -------- | ----------------------------------------------------------------------- |
| **Invisible/Intangible** | Makes the opponent’s ball invisible          | Strong     | Considered very powerful/unbalanced but fun; requires balancing |
| **Ink Blot**        | Obscures the opponent’s vision              | Medium    | Mario Kart style; blocks part of the screen                        |
| **Bumper reduction**    | Reduces the size of opponents’ bumpers            | Medium    | Makes scoring more difficult                                          |
| **Black hole**           | Spawns a hole at a random position             | Strong     | Considered too punitive; needs balancing or removal                             |
| **Modify bounce**   | Increases or randomises the ball’s bounce | Variable | Disrupts predictable trajectories                                   |
| **Sticky bumpers**     | The ball sticks to the bumpers                       | Medium    | Requires spamming the flippers to break free                         |
